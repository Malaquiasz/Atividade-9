Reflexão e Aplicação Prática

  • display: flex; - Ativa o modelo Flexbox para o contêiner.
  • flex-direction - Define a direção dos itens flexíveis (linha ou coluna).
  • justify-content - Alinha os itens ao longo do eixo principal.
  • align-items - Alinha os itens ao longo do eixo transversal.
  • flex-wrap - Controla se os itens devem ser quebrados em várias linhas.

Aplicação Prática

Para um menu de navegação responsivo, eu usaria Flexbox para:

  • Definir o contêiner do menu como display: flex; para alinhar os itens horizontalmente.
  • Utilizar justify-content: space-between; para distribuir o espaço entre os itens do menu.
  • Aplicar flex-wrap: wrap; para garantir que o menu se ajuste em telas menores.
